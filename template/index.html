A World of Webmaker Apps.

This is a prototype.

Webmaker Connect is an experiment that makes it possible for third-parties and mad scientists to create Web applications that securely integrate with Mozilla Webmaker on a user's behalf.

Currently, the only apps that exist in the Webmaker universe are Thimble and Popcorn Maker. But what if more apps could be part of the ecosystem? What if your Twine, Meemoo, and Waterbear creations could appear in your Webmaker profile, with invitations for others to build upon and remix them? What if users could seamlessly "level up" from Thimble to a more advanced tool like jsfiddle, jsbin, or Brackets without leaving Webmaker behind?

Webmaker Connect is an attempt to explore that possibility. At its core, it's an OAuth provider that provides APIs for third-party applications to request and obtain access to a user's Webmaker account. At the moment, applications can only request access to a Webmaker's username, user ID, and email hash (which is useful for displaying gravatars and issuing open badges). But more is on the way.

For more information, please see the use cases etherpad and the github repo.
